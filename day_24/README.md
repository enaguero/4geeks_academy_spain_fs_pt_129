# 🧱 Día 24: Data Structures (OOP + Family Static API)

## 🎯 Objetivos de Aprendizaje

Al terminar este día deberías poder:

- Explicar qué es Programación Orientada a Objetos (OOP) en Python.
- Diferenciar `class`, `object`, `attribute` y `method`.
- Modelar una estructura de datos simple usando clases.
- Exponer esa estructura desde una API en Flask.
- Prepararte para el salto a modelado con base de datos (Día 25+).
