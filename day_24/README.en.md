# 🧱 Day 24: Data Structures (OOP + Family Static API)

## 🎯 Learning Objectives

By the end of this day you should be able to:

- Explain what Object-Oriented Programming (OOP) is in Python.
- Distinguish between `class`, `object`, `attribute`, and `method`.
- Model a simple data structure using classes.
- Expose that structure through a Flask API.
- Get ready for the jump to database modeling (Day 25+).
