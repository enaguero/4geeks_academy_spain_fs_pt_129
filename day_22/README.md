# 🐍 Día 22: Intro a Python

## 🎯 Objetivos de Aprendizaje

Hoy comienzas el módulo de backend con Python. Al terminar este día deberías poder:

- Entender qué hace un desarrollador backend en una aplicación real
- Leer y escribir sintaxis básica de Python
- Tomar decisiones con condicionales (`if`, `elif`, `else`)
- Trabajar con listas para almacenar y manipular colecciones
- Crear funciones reutilizables con parámetros y `return`
- Resolver pequeños problemas combinando todo lo anterior

---

## 🔗 De JavaScript a Python (Mapa Mental Rápido)

```
Variables y tipos
    ↓
Condicionales
    ↓
Listas
    ↓
Loops
    ↓
Funciones
    ↓
Resolver problemas completos
```

Si vienes de JavaScript, piensa en Python como una versión más legible y directa para lógica de backend.

## 📚 Estructura del Día

Este día tiene **8 pasos progresivos**. Sigue el orden.

### Step 0: Contexto Backend + Python 🧭
**Carpeta**: `step0-backend-python-contexto/`

Antes de programar, entenderás el rol backend y por qué Python es una herramienta fuerte para empezar.

**Conceptos**:
- Frontend vs Backend
- Ciclo request/response
- Por qué Python en backend y automatización
- Preparar entorno mínimo (`python3`)

---

### Step 1: Sintaxis y Fundamentos ✍️
**Carpeta**: `step1-sintaxis-fundamentos/`

Tu primer código en Python: variables, tipos y operaciones básicas.

**Conceptos**:
- Reglas de sintaxis e indentación
- Variables y tipos (`str`, `int`, `float`, `bool`)
- `print()` e `input()`
- Conversión de tipos

---

### Step 2: Condicionales en Python 🔀
**Carpeta**: `step2-condicionales-python/`

Tomar decisiones con lógica booleana.

**Conceptos**:
- `if`, `elif`, `else`
- Operadores de comparación
- `and`, `or`, `not`
- Errores comunes de lógica

---

### Step 3: Listas en Python 📦
**Carpeta**: `step3-listas-python/`

Guardar colecciones de datos y modificarlas.

**Conceptos**:
- Crear listas
- Índices y slicing
- Métodos (`append`, `remove`, `pop`, `sort`)
- Listas de strings y números

---

### Step 4: Loops + Listas 🔁
**Carpeta**: `step4-loops-listas/`

Recorrer colecciones y repetir tareas.

**Conceptos**:
- `for` con listas
- `while` para repeticiones condicionadas
- `range()` y `enumerate()`
- `break` y `continue`

---

### Step 5: Funciones en Python 🧩
**Carpeta**: `step5-funciones-python/`

Encapsular lógica para reutilizar código.

**Conceptos**:
- Definir funciones con `def`
- Parámetros y argumentos
- `return`
- Scope básico de variables

---

### Step 6: Mini Retos Integradores 🧪
**Carpeta**: `step6-mini-retos/`

Aplicar condicionales + listas + loops + funciones en problemas reales pequeños.
