# 🚀 Step 7: Plan de Práctica Interactiva

## 🎯 Objetivo

Consolidar lo aprendido hoy en plataformas interactivas oficiales, en un orden que maximice comprensión.

---

## 🛣️ Ruta recomendada (en orden)

1. [Learn Python Interactively (beginner)](https://4geeks.com/syllabus/spain-fs-pt-129/practice/python-beginner-exercises)
2. [Learn Python Functions Interactively](https://4geeks.com/syllabus/spain-fs-pt-129/practice/python-function-exercises)
3. [Learn Python Loops and lists Interactively](https://4geeks.com/syllabus/spain-fs-pt-129/practice/python-loops-lists-exercises)
4. [Master Python by practice (interactive)](https://4geeks.com/syllabus/spain-fs-pt-129/practice/master-python-exercises)

---

## ⏱️ Timeboxing sugerido

- Bloque 1 (30-40 min): Beginner
- Bloque 2 (25-35 min): Functions
- Bloque 3 (25-35 min): Loops + Lists
- Bloque 4 (30-45 min): Master practice

Si no te da tiempo hoy, completa al menos los primeros 3 bloques.

---

## ✅ Criterios para pasar al Día 23 (API con Flask/FastAPI)

Antes de avanzar, asegúrate de que puedes:

- Escribir un `if/elif/else` sin ayuda
- Recorrer una lista y filtrar datos
- Crear funciones que devuelvan resultados
- Resolver un ejercicio pequeño de punta a punta

---

## 🧰 Método de estudio recomendado

1. Intenta resolver sin mirar solución
2. Si te bloqueas, busca una pista, no la respuesta completa
3. Cuando lo logres, reescribe la solución desde cero
4. Explica en voz alta qué hace tu código (Feynman)

---

## 🔁 Si te atascas

Haz este protocolo:

1. Reduce el problema al caso más pequeño posible
2. Imprime variables para ver su valor real
3. Verifica tipos con `type()`
4. Revisa indentación y condiciones límite
5. Vuelve a ejecutar

---

## 🎯 Meta final del día

Terminar con confianza para escribir scripts de Python básicos y entrar al Día 23 con base sólida para crear tu primera API con Flask o FastAPI.
