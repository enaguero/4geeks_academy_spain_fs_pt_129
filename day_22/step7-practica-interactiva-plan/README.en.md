## ✅ Criteria to move on to Day 23 (API with Flask/FastAPI)

Before moving on, make sure you can:

- Write an `if/elif/else` without help
- Iterate over a list and filter data
- Create functions that return results
- Solve a small exercise end to end

---

## 🧰 Recommended study method

1. Try to solve without looking at the solution
2. If you get stuck, look for a hint, not the full answer
3. Once you get it, rewrite the solution from scratch
4. Explain out loud what your code does (Feynman)

---

## 🔁 If you get stuck

Follow this protocol:

1. Reduce the problem to the smallest possible case
2. Print variables to see their actual value
3. Verify types with `type()`
4. Check indentation and edge conditions
5. Run it again

---

## 🎯 Final goal of the day

Finish with the confidence to write basic Python scripts and start Day 23 with a solid foundation for building your first API with Flask or FastAPI.
