# 📋 Day 3: HTML Forms

## 📚 Context

Learn how to build HTML5 forms: inputs, textareas, native validations, and how to handle the information the user submits.

## 🎯 Syllabus resources

- **READ** – [Understanding HTML Input HTML Text Area and Forms](https://4geeks.com/syllabus/spain-fs-pt-129/read/html-input-html-textarea)
- **PRACTICE** – [Learn how to use and interact with HTML Forms](https://4geeks.com/syllabus/spain-fs-pt-129/practice/forms-exercises)
- **PROJECT** – [Create a HTML5 form](https://4geeks.com/syllabus/spain-fs-pt-129/project/html5-form)

> 💡 There's no local exercise folder for this day — all material is in the links above.

## ✅ Day checklist

- [ ] I know the most common `<input>` types (text, email, password, number, date, checkbox, radio)
- [ ] I can use `<textarea>`, `<select>`, and `<option>`
- [ ] I understand the `required`, `pattern`, `min`, `max`, `placeholder` attributes
- [ ] I can link a `<label>` to an `<input>` using `for` and `id`
- [ ] I completed the HTML5 form project
